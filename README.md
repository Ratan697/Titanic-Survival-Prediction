# Titanic Survival Prediction 🚢

This project uses Machine Learning techniques to predict passenger survival on the Titanic dataset. The workflow includes data cleaning, handling missing values, exploratory data analysis (EDA), feature engineering, model training, and evaluation.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature encoding and scaling
- Model training using Scikit-learn
- Performance evaluation with accuracy metrics
- Survival prediction for new passenger data

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Goal
To build a classification model that accurately predicts whether a passenger survived the Titanic disaster based on available passenger information.
